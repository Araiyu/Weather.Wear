#ツールライブラリの名前
「Weather.Wear」(ウェザー.ウェアー)
	Weather.Wearはweather天気とWaer服を身につけるという天気を身につけると言った単語からこのサイト名にしました。
	このサイトは、その日の天気によってどんな服装があっているのか自分で決めてコーディネートを選ぶことの参考にしてほしいと思い制作しました。
